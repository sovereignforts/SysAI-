# Contributing to NLP Chatbot

We appreciate your interest in contributing! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/nlp-chatbot.git`
3. Create a virtual environment: `python -m venv venv`
4. Install dependencies: `pip install -r requirements.txt`
5. Create a feature branch: `git checkout -b feature/your-feature`

## Development Guidelines

### Code Style
- Follow PEP 8 style guide
- Use type hints for functions
- Write descriptive docstrings
- Keep functions focused and small

### Testing
- Write tests for new features
- Run tests: `pytest tests/`
- Aim for >80% code coverage

### Commits
- Use clear, descriptive commit messages
- Follow: `type: subject` format
  - `feat:` for new features
  - `fix:` for bug fixes
  - `docs:` for documentation
  - `test:` for tests
  - `refactor:` for code refactoring

## Pull Request Process

1. Update README.md with any new features
2. Update requirements.txt if adding dependencies
3. Write/update tests for your changes
4. Ensure all tests pass: `pytest`
5. Check code quality: `flake8 src/`
6. Submit PR with detailed description

## Areas to Contribute

- **Models**: Improve NLP, vision, and mood detection models
- **Features**: Add new capabilities (voice support, persistence, etc.)
- **Documentation**: Improve docs, add examples
- **Testing**: Increase test coverage
- **Performance**: Optimize code and models
- **Fixes**: Bug reports and fixes

## Questions?

- Open an issue for discussions
- Check existing issues before posting
- Be respectful and constructive

## Code of Conduct

- Treat all contributors with respect
- No discrimination or harassment
- Focus on what's best for the community

Thank you for contributing!
